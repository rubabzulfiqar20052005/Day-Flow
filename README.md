# 📅 DayFlow
### *Your Day. Your Habits. Your Flow.*

> A beautifully designed Android app to track daily activities, build habits, write notes — all organized around your calendar.

<br>

</div>

---

## ✨ Features

### 📆 Calendar-Based Activity Tracking
- View and manage your daily activities organized by date
- Navigate easily through days, weeks, and months
- Mark activities as complete with a single tap

### 🔁 Habit Tracker
- Create custom habits and track your streaks
- Daily habit check-ins linked directly to your calendar
- Visual progress indicators to keep you motivated

### 📝 Notes
- Write personal notes for any day
- Quick-access notes tied to your schedule
- Rich text support for better organization

### 🎨 Advanced UI & Animations
- Smooth, fluid animations throughout the app
- Modern Material Design with custom components
- Intuitive and elegant user interface built for daily use

### ⚙️ Settings
- Personalize your app experience
- Notification and reminder preferences
- Theme and display customization options

### 💬 Feedback System
- Share your experience directly from the app
- Built-in feedback form for suggestions and bug reports
- Helps improve DayFlow continuously

### Prerequisites
- Android Studio (latest version recommended)
- Android SDK 21+
- Java / Kotlin

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/rubabzulfiqar20052005/Day-Flow.git
   ```

2. **Open in Android Studio**
   ```
   File → Open → Select the Day-Flow folder
   ```

3. **Sync Gradle**
   ```
   Let Android Studio sync all dependencies automatically
   ```

4. **Run the app**
   ```
   Connect your device or use an emulator → Click ▶ Run
   ```
